This is the iBike Project folder.
Developed with Netbeans 8.0.2, tested with Java JDK 1.8, Glassfish 4.1 and MySQL Workbench CE 6.3.

To deploy and run the project :
	- Use the SQL script to create the database,
	- Make sure connectivity to the database (MySQL) is insured in Netbeans,
	- Insert data into the database thanks to "iBikeDatabaseData.txt",
	- Start Glassfish server,
	- Deploy and run the project.
